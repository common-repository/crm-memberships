== Description ==

CRM Memberships plugin allows restricting your content to paid or registered members only. Use it for creating online courses, marketing funnel fulfillment etc. CRM Memberships plugin also allows easy integration of WordPress with CRMS such as Salesforce.

### Features 

* <strong>Publications</strong>: Create courses and publications. 
* <strong>Content Protection</strong>: Allows restricting your content to paid or registered members only.
* <strong>Access Tags</strong>: Tag based permissions. Add permission tags to any posts, pages, categories or user profiles! 
* <strong>Publication Wizard</strong>: Create publications easily and associate access tags. 
* <strong>Publication Gallery</strong>: Out of the box gallery page that lists all active publications. 
* <strong>Compatibility</strong> Works with your existing contents.
* <strong>Measure User Engagement</strong>: Identify popular publications based on user actions. 
* <strong>Subscriber Reports</strong>: List subscribers by Access Tags. 
* Works with Salesforce CRM using our ([Managed Package for Salesforce](http://netesenz.com/)).
* Integration & Migration Help. [Get it](http://netesenz.com/contact-us.html/) for importing data from other CRM and membership products.
* Priority Support. [Get it](http://netesenz.com/contact-us.html/) Priority support plan will help you get the help from a dedicated team.
* <strong>CRM Integration</strong> We are going to add more CRM support in the future. You can request the one you want and we will add it for you on a priority basis! 
* <strong>Easy to use</strong> Works out of the box with Minimal Settings

== Installation ==

* Upload `crm-membership` plugin to the `/wp-content/plugins/` folder and Activate it.
* It will add a menu 'CRM Memberships' in the admin panel in the left menu.

== Changelog ==

= 1.0 =
*Release Date - 03 January 2021*

* The very first release of plugin. Add a menu 'CRM Memberships' at the left menu of the admin panel.
* In the Settings tab, admin can set Member Login Page Logo, Membership Publication page path, Enable Log, Salesforce NTZ CRM API Key, Token & User Settings etc.
* Under Publication Wizard Digital products can be created along with defining their Title, Enable & Disable images, & To Manage these digital products Access conditions Access Tags can be assigned to them. Also each publication shortcode can be copied from here.
* The Publication tab returns the list of All Digital Products registered.
* Access Tags defines & controls the access permission, that can be assigned to publication before assign it to any Subscriber
* Subscriber Tab returns the list of members along with the list of publications they have registered.
